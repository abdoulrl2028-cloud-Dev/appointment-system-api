# appointment-system-api
O que o sistema faz  Cadastro de usuários  Cadastro de serviços  Agendamento de horários  Listagem paginada  Validações  Tratamento global de erros  Documentação automática com Swagger




Yes, I have actively committed code to GitHub, including a complete backend project called Appointment System API, built using Java and Spring Boot.

In this project, I implemented:

User registration and management

Service registration

Appointment scheduling system

Pagination for listing data

Input validation

Global exception handling

Automatic API documentation using Swagger

Docker and Docker Compose configuration

The project follows a structured layered architecture and uses Maven for dependency management.
I regularly commit features, configuration updates, documentation improvements, and Docker setup. I use Git for version control with organized commits and proper project structure.
